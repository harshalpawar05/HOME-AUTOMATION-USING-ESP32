# "HOME AUTOMATION USING ESP32"

A smart home automation system that allows users to control home appliances using ESP32 and Google Assistant voice commands.  
The main concept behind this project is IoT-based wireless home automation using Wi-Fi communication.

---

## Components used

1. ESP32 DevKit V1
2. 4-Channel Relay Module
3. Push Buttons
4. AC Bulbs / Appliances
5. Jumper Wires
6. Power Supply
7. Wi-Fi Connection

## Features

- Control appliances using Google Assistant
- Wireless home automation using Wi-Fi
- Real-time appliance control
- ESP32 web server integration
- Manual switch control
- Low-cost smart home solution

---

## Technologies Used

- ESP32
- Arduino IDE
- Google Assistant
- IoT

---

## Working

1. ESP32 connects to Wi-Fi.
2. Google Assistant sends command.
3. ESP32 receives the command.
4. Relay module switches appliances ON/OFF.
5. Appliances can also be controlled manually using push buttons.

---

## Voice Commands

- "Turn on the light"
- "Turn off the light"
- "Switch on bedroom light"
- "Switch off appliances"
